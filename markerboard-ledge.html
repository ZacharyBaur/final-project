Markerboard Ledge

2227 Spikes St, Lansing, MI 48906

This is a rather tall ledge featuring angle iron coping. Lots of smooth run up and run off makes this spot a must-see when skating through Mid-Michigan. If you try and skate this spot during weekdays and business hours it is likely a bust. This is a spot best saved for weekends and after-hours.
